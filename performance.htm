Training Data Performance

As mentioned in the About page, the classifier was trained using a simple random forest classifier on 80% of the provided training data. After test on the remaining 20% of data, an accuracy metric of 0.7805 was generated. Although this accuracy metric seems rather high, the confusion matrix shown below shows that the classifier is much more reliable for predicting if there isn't an alpha helix at a protein than if there is. A corresponding ROC curve with an AUROC value of 0.72 is shown below:

TN = 58974 FP = 289
FN = 16292 TP = 108

Testing Data Performance

As for the provided testing data, an accuracy metric of 0.7878 was generated, with the corresponding confusion matric and ROC curve shown below. The AUROC value was only slightly lower than that that of the training data at 0.71, with similar proportions in the confusion matrix, which indicates that there is consistency with the classifier:
