Error Message

If the UniProt ID input leads to an error page, do the following:

  1. Visit the official website here and confirm that the ID exists and has a page with a .FASTA link containing the amino acid sequence.
  2. Click on the link and obtain the ID XXXXX directly from the link: https://www.uniprot.org/uniprot/XXXXX.fasta
  3. Reload the predictor homepage and input the ID.
